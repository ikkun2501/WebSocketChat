WebSocketChat
====
WebSocketを利用したチャットウェブアプリケーションです。
フロントエンドをElm、サーバをSpringのWebFluxで実装しています。

## Requirement
* npm
* JDK 8
## Install

### front end
```
$ npm install
```

## Usage

### fornt side
```
$ npm run dev
```
### server side
```
$ ./gradlew bootRun
```

### browser
```
http://localhost:3000
```
